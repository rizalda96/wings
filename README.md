## Cara instalasi aplikasi

- git clone https://github.com/rizalda96/wings.git
- untuk setingan ENV bisa dicopy dari ENV.EXAMPLE
- composer install
- npm install
- php artisan migrate
- php artisan db:seed

## Cara menjalankan aplikasi
- php artisan serve
- npm run watch
